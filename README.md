# Goldmark

Full-stack e-commerce platform for jewellery sales. Built as a college capstone project with React frontend and Node.js backend.

## Tech Stack

**Frontend (goldmark/):**
- React
- Vite
- TypeScript
- Tailwind CSS
- Zustand (state management)

**Backend (goldmark-backend/):**
- Node.js + Express
- PostgreSQL
- Supabase Auth
- Stripe (payment processing)

## Features

- Product catalog with search and filtering
- Shopping cart and checkout flow
- User authentication (Supabase)
- Payment processing (Stripe integration)
- Order management
- Admin dashboard

Built end-to-end as a college project to demonstrate full-stack development skills across the entire e-commerce flow.
